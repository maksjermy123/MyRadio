# 📖 Оглавление + 📻 Радио — Telegram Mini App

Христианский канал «От чтения к Преображению» как Telegram Mini App:
- Оглавление всех постов канала по категориям с поиском
- Христианское радио (несколько станций, прямой эфир)
- Проверка подписки на канал перед входом
- Автоматическая индексация новых постов через webhook

---

## Архитектура проекта

```
Telegram-канал (@Chtenie_Preobrazenie)
        │
        │  новый пост с #хэштегом
        ▼
Render (main.py — FastAPI)
        │
        ├── проверяет хэштег → определяет категорию
        ├── скачивает posts.json с GitHub
        ├── добавляет пост → загружает обратно
        │
        ▼
GitHub (posts.json — хранилище данных)
        │
        ▼
GitHub Pages (index.html — Mini App)
        │
        ▼
Пользователь открывает Mini App → видит обновлённое Оглавление
```

## Система хэштегов → категории

Каждый пост в канале должен содержать хотя бы один хэштег из таблицы ниже. Пост без известного хэштега не попадает в оглавление.

| Хэштег в канале | Категория в оглавлении |
|---|---|
| `#библия` | 📖 Библия и толкование |
| `#богословие` | ✝️ Богословие |
| `#теодицея`, `#фильм` | 😔 Теодицея |
| `#книги`, `#достоевский`, `#солженицын`, `#клайвльюис`, `#чехов`, `#лесков`, `#толстой` | 📚 Книги и авторы |
| `#достоевский` | 📚 Достоевский |
| `#жизнь` | 🌱 Христианская жизнь |
| `#молитва` | 🙏 Молитва и духовная жизнь |
| `#духовныйдневник` | 📔 Духовный дневник |
| `#проповедь`, `#семинар` | 🎤 Проповедь и семинар |
| `#челлендж`, `#лука` | 📅 Челлендж: Лука |
| `#история` | 🏛️ История и церковь |
| `#размышления`, `#цитата` | 💬 Размышления и цитаты |
| `#юмор` | 😄 Юмор |
| `#праздник` | 🎄 Праздники |
| `#анонс`, `#новости` | 📻 Анонсы канала |
| `#отчтениякпреображению` | *(служебный — игнорируется)* |

Один пост может иметь несколько хэштегов → попадёт в несколько категорий.
Если отредактировать пост в канале и добавить/изменить хэштег — оглавление обновится автоматически.

## Первоначальная настройка (с нуля)

### Шаг 1 — Создать бота в Telegram

1. Открой **@BotFather** → `/newbot`
2. Придумай имя и username (должен заканчиваться на `bot`)
3. Сохрани токен вида `1234567890:AAG...`
4. `/mybots` → выбери бота → **Bot Settings → Group Privacy → Disable** (иначе бот не видит посты)
5. Добавь бота администратором в канал (право «Управление сообщениями» обязательно)

### Шаг 2 — Создать GitHub Token

1. github.com → аватар → **Settings → Developer settings → Personal access tokens → Tokens (classic)**
2. **Generate new token** → название `MyRadio` → срок **No expiration** → галочка ✅ `repo`
3. Сохрани токен `ghp_...` — показывается только один раз!

### Шаг 3 — Задеплоить бэкенд на Render

1. Зайди на **render.com** → **New Web Service**
2. Подключи репозиторий `maksjermy123/MyRadio`
3. Runtime: **Python 3**, Start Command: `uvicorn main:app --host 0.0.0.0 --port $PORT`
4. Вкладка **Environment** → добавь все переменные из таблицы выше
5. После деплоя скопируй URL вида `myradio-xxxx.onrender.com`

### Шаг 4 — Зарегистрировать webhook

Открой в браузере:
```
https://myradio-xxxx.onrender.com/set_webhook
```
Должен увидеть `"ok": true`. Повторяй после каждого изменения списка `allowed_updates`.

### Шаг 5 — Настроить кнопку меню бота

В **@BotFather**:
```
/mybots → выбери бота → Bot Settings → Menu Button → Configure menu button
URL:   https://t.me/preoradio_bot/radio
Текст: 📖 Оглавление + Радио
```

### Шаг 6 — Закреплённый пост в канале с кнопкой

Отправить пост через Bot API (вставь токен):
```
https://api.telegram.org/botВАШ_ТОКЕН/sendMessage?chat_id=@Chtenie_Preobrazenie&text=📻%20Христианское%20радио%20—%20слушай%20прямо%20здесь&reply_markup={"inline_keyboard":[[{"text":"📖 Оглавление + Радио","url":"https://t.me/preoradio_bot/radio"}]]}
```
Закрепи появившийся пост в канале.

## Поддержание сервиса

**Render засыпает** на бесплатном тарифе через 15 минут без запросов. Решение — **UptimeRobot**:
- Мониторит `https://myradio-rrsk.onrender.com/` каждые 5 минут
- Render не засыпает → webhook всегда готов принять новый пост

**GitHub Token** — создан без срока истечения, но если скомпрометирован — немедленно удали в настройках GitHub и создай новый. Никогда не публикуй токен в открытом чате или коде.

**Логи Render** — при проблемах с индексацией открой Render → Logs и ищи строки:
```
▶ update_id=XXXXX | поля: ['channel_post']
Пост 359 | теги: ['#библия'] | темы: ['📖 Библия и толкование']
Пост 359 — added ✓
```
